About Us

Welcome to the Ascii Generator! This project was created to provide a simple and interactive way for users to generate ASCII art from any text they enter.

Project Overview

The Ascii Generator lets you input any text and select a banner style (such as Standard, Shadow, or Thinkertoy). Once you submit the form, the app processes the input text and generates the corresponding ASCII art in the selected style, which is then displayed on the page.

Technology Stack

  • Backend: Go (Golang) - For the server-side logic and ASCII art generation.
  • Frontend: HTML, CSS - For the user interface.
  • Template Engine: Go's `html/template` package - Used for rendering HTML pages with dynamic content.
  • File Handling: Go's `os` and `strings` packages - Used to load and process ASCII banner files and manage input text.
